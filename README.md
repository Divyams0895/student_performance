# 🎓 Student Performance Analytics Dashboard

An interactive **Power BI dashboard** designed to analyze student performance, teacher effectiveness, and subject-wise insights. The dashboard helps educators and decision-makers identify academic trends, monitor performance, and make data-driven decisions.

## 📊 Dashboard Overview

The dashboard consists of **three interactive pages**:

### 1️⃣ Overview
- Total Students
- Total Teachers
- Total Subjects
- Average Score
- Pass & Fail Analysis
- Grade Distribution
- Student Performance KPIs

### 2️⃣ Student Analysis
- Student-wise Performance
- Score Distribution
- Grade Analysis
- Performance Trends
- Interactive Filters

### 3️⃣ Teacher & Subject Insights
- Total Teachers & Subjects
- Average Subject Score
- Average Teacher Rating
- Best Performing Subject
- Average Score by Subject
- Teacher-wise Average Performance
- Students per Teacher
- Top 5 Teachers by Average Score
- Bottom 5 Subjects by Average Score
- Dynamic Insight Card using DAX
